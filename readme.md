
# Advent of Code 2025 — Soluções em Elixir

Repositório com soluções dos desafios do **Advent of Code** ([adventofcode.com](https://adventofcode.com/)) implementadas em **Elixir**. Cada dia fica em `dayNN/` com scripts (`part01.exs`, `part02.exs`) e arquivos de entrada; foco em soluções concisas e comentadas para praticar a linguagem.
